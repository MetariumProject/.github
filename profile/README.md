## metarium

sovereignty in digital exploration

---

metarium is a trellis — a quiet framework for digital explorers.

in the physical world, sketches, prototypes, and discarded attempts form natural trails of discovery. in the digital world, these artifacts are fragile — easily erased, lacking the contextual layers that give their physical counterparts meaning.

metarium uses blockchain to create sovereign timelines — for self-validation and trustless proof of events. it preserves the deliberate act of creation and the stillness of moments between.

it is not a product. it is a foundation.

---

### kuri (குறி)

the fundamental unit — a static inscription on the blockchain.

a sign. a stamp. a marker. a symbol. an indication. a designation.

each kuri is permanent and context-free. the multiplicity of meaning is deliberate.

### mai (மை)

ink. the native token that writes on the blockchain.

each metarium instance creates its own mai — independent and unique. anyone can deploy a metarium and create their own ink.

---

### built on substrate

metarium is a blockchain template built on [substrate](https://substrate.io) by the [web3 foundation](https://web3.foundation) ecosystem. what we have built is a pallet on top of substrate — reduced to its essentials, open to evolving needs.

- substrate cryptographic primitives (sr25519 / ed25519)
- polkadotjs-compatible encrypted key format
- argon2id password hashing
- each deployment is an independent chain

---

### get started

the fastest way to begin is through **[metarium.ai](https://metarium.ai)** — point any ai coding agent at the site and it will guide you through setup autonomously.

```
git clone https://github.com/MetariumProject/metarium-ai.git
```

three levels of engagement:

- **connect** — link to an existing cms and start creating kuris
- **deploy a cms** — run your own content management layer on google cloud
- **deploy a chain** — full substrate node with the metarium runtime for complete sovereignty
